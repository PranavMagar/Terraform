# Beginner-Friendly Tasks

Here are 10 tasks to help you learn and improve the project:

1. **Add "Edit Profile" Page**: Create a route and template to allow users to change their password.
   - *Skills*: FastAPI Forms, SQL Update.

2. **Implement Pagination**: Add pagination to the dashboard expense list (e.g., 10 per page).
   - *Skills*: SQL Limit/Offset, Jinja2 Logic.

3. **Add "Export to CSV"**: Add a button to download expenses as a CSV file.
   - *Skills*: Pandas or CSV module, File Response.

4. **Docker Network Isolation**: Modify `docker-compose.yml` to put the DB in a private network, accessible only by the web app.
   - *Skills*: Docker Networking.

5. **Add Request Logging**: Middleware to log every request method, URL, and processing time.
   - *Skills*: FastAPI Middleware, Python Logging.

6. **Terraform S3 Bucket**: Add an S3 bucket resource in Terraform to store user uploads (e.g., receipts).
   - *Skills*: Terraform Resources.

7. **Jenkins Email Notification**: Add a step in `Jenkinsfile` to send an email on build failure.
   - *Skills*: Jenkins Plugins, Groovy.

8. **Add "Budget" Feature**: Allow users to set a monthly budget and show a progress bar.
   - *Skills*: DB Schema Change, Frontend Logic.

9. **Unit Test for Reports**: Write a test case for the `/api/chart-data` endpoint.
   - *Skills*: Pytest, JSON Assertions.

10. **Refactor to Async**: Convert database calls to use `async/await` with `asyncpg`.
    - *Skills*: AsyncIO, SQLAlchemy Async.
